# Requirements

- Globbaly installed expo ( `yarn global add expo`)

# Setup

1. First it is necessary to run the project logged in expo via terminal using:
  ``` 
    expo login
  ```

2. Run the `node_heat` backend:
  ```
    $ cd ../
    $ cd node_heat
    $ yarn dev
  ```

3. Use the credential obtained in the GitHub OAuth app to fill the new `.env` file following the `.env.example` pattern in the project `node_heat`.

4. After that: 
  ``` 
    yarn start
  ```
